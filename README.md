### 제 1차 자바 프로젝트


### 기쁘띠          쁘띠 조
#### 기프티콘 거래 중개 웹 페이지


---
중앙인재개발원 클라우드 데브옵스 프론트엔드&백엔드 자바 풀스택 개발자 취업캠프 


---
### 주제 설명
관리자가 카테고리를 설정하여 판매 가능한 상품을 등록하면
사용자가 등록된 상품 중에 선택하여 판매 상품을 등록하고
사용자가 기프티콘을 구매/판매 할 수 있도록 중개하는 시스템
